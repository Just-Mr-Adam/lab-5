Менеджер паролей (локально) — простое консольное приложение на C# для безопасного хранения учётных записей без использования облака или сети.
Программа реализует базовые CRUD-операции: создание, чтение, обновление и удаление записей (сайт/сервис, логин, пароль и заметки).
Все данные хранятся локально в зашифрованном JSON-файле, пароли шифруются с помощью симметричного алгоритма AES перед сохранением.
Для доступа к менеджеру используется мастер-пароль, который вводится при запуске и используется как ключ шифрования.
Приложение не требует установки внешних зависимостей и работает полностью оффлайн, обеспечивая базовый уровень защиты паролей на локальном устройстве.
